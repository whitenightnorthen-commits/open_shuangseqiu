# 双色球助手

这是一个用于双色球历史数据查看、趋势统计、号码记录和辅助筛选的安卓 App。

> 彩票开奖结果具有随机性，本软件仅用于数据记录、辅助筛选和回顾复盘，不保证中奖，请理性使用。

## 下载

请到本仓库的 **Releases** 页面下载最新版 APK：

- 文件名：`ssq-helper-v1.1.apk`
- 当前版本：`v1.1`
- 安卓安装包：APK

如果你手机里已经安装过旧版，请直接覆盖安装新版，不要先卸载。覆盖安装可以保留本地数据，例如 API Key、记录本内容和历史设置。

## 主要功能

- 查看双色球历史开奖数据
- 联网更新最新开奖记录
- 红球、蓝球走势和统计分析
- 策略推荐号码，按评分排序，仅供辅助筛选
- 点评号码，结合本地统计信息进行分析
- 投注记事本，可记录倍数、是否已购买，并在开奖后回顾命中情况

## 数据与隐私

- App 数据主要保存在手机本地。
- 用户填写的 API Key 保存在本机，不会上传到本仓库。
- 本仓库仅用于发布 APK，不公开项目源码。

## 安装说明

1. 在 Releases 页面下载最新版 APK。
2. 在安卓手机中打开 APK。
3. 如果系统提示“未知来源应用”，请按提示允许安装。
4. 安装完成后打开“双色球助手”。

## 更新说明

新版 APK 与旧版使用相同包名和签名，可直接覆盖安装旧版并保留本地数据。

如需升级，请下载最新 APK 后直接安装。
